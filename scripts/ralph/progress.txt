## Codebase Patterns
- ESLint config: uses `eslint.config.mjs` flat config with `eslint-config-next` v16 (exports flat config directly, no FlatCompat needed)
- `next lint` works but shows deprecation warning for Next.js 16; can also use `npx eslint` directly
- `lib/hooks/` contains utility hooks; the fluid cursor hook uses `@ts-nocheck` and `export default` (exception to named export convention due to third-party origin)
- Import `useFluidCursor` as `fluidCursor` (or any non-`use` prefix) to avoid ESLint react-hooks/rules-of-hooks error when calling inside useEffect
- Always guard `useFluidCursor` against null canvas and null WebGL context ‚Äî environments without WebGL (headless browsers, some mobile) will fail silently
