## Codebase Patterns
- Footer is a named export in `app/components/footer.tsx`, used by PageLayout
- All bilingual components accept `lang?: "zh" | "en"` prop with `useLocale()` from next-intl as fallback
- Use `Link` from `@/i18n/navigation` for locale-aware links (auto-prefixes locale)
- Components using next-intl hooks (`useLocale`, `usePathname`) need `"use client"` directive
- AnimatedSignature is a default export (exception to the named export convention)
- Nav links need `relative z-10` to be clickable over any background effects
- Content JSON files in `content/` use `title`/`titleEn` pattern for bilingual data
