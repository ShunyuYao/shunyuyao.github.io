## Codebase Patterns
- Content sections (Work, Papers, Press, Awards, Archive) all use `SectionContainer` wrapper with `group` hover for section-level background highlight
- Individual list items in sections are `<li>` elements — use `px-3 py-2 -mx-3 rounded-md` with negative margin to add hover padding without shifting layout
- Section-level hover is `bg-neutral-900` full background; item-level hover uses `bg-neutral-800/50` for subtle contrast
- NowSection uses `list-disc` bullet style, separate from the card-style grid sections
- Stale `.next/types/*.d 2.ts` files may appear — safe to delete before typecheck
- Stale `out/` directories with spaces (e.g., `en 2`, `_next 2`) can break build — clean `out/` before rebuild
- Use CSS `mask-image` gradient for text fade-out effects — background-agnostic and works with hover states without extra handling
